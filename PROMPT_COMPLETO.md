# Prompt Completo Para Seguir Mejorando El Sistema

```text
Actuá como arquitecto senior full-stack, especialista en sistemas gastronómicos, React, TypeScript, Vite, Tailwind CSS, Supabase/PostgreSQL, UX operativa para restaurantes, caja, tickets PDF, cocina, inventario y escandallos.

Revisá el proyecto completo “El Patrón - Sistema Gestor Gastronómico” y mejorá todo lo necesario para dejarlo más profesional, estable, rápido, seguro y usable en restaurante real.

Prioridades:
1. Verificar TypeScript, build, imports, tipos débiles y estados inconsistentes.
2. Revisar flujo de pedidos: mozo, cocina, descuento de stock, caja, facturación y mesa libre.
3. Revisar inventario: insumos, bodega, recetas, mermas, movimientos y alertas.
4. Revisar caja: cobro, pagos mixtos, PDF, impresión/ticketera y cierre diario.
5. Revisar Supabase: variables de entorno, migración SQL, servicios, errores, RLS y fallback offline.
6. Mejorar UX táctil para salón, cocina, caja y administración.
7. Mantener identidad visual de El Patrón y el logo institucional.
8. No eliminar funcionalidades existentes sin reemplazo claro.
9. Verificar con `npm run lint` y `npm run build`.

Entregable:
- Cambios implementados.
- Archivos modificados.
- Validaciones ejecutadas.
- Pendientes para producción.
```
